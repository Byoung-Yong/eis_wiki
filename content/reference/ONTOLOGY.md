# Project ontology

## Node types

Define the node types used by this project. Start with `Concept`, `Entity`, `PhysicalQuantity`, `Observable`, `ModelParameter`, `TheoreticalConstruct`, `Law`, `Equation`, `ModelOrTheory`, `Approximation`, `Process`, `Mechanism`, `Condition`, `ExperimentalMethod`, `ComputationalMethod`, `TopicMap`, `DomainMap`, `GraphView`, and `Page` as needed.

## Relation types

Define every allowed relation, its direction, and an example. Prefer specific relations over `RELATED_TO`.
