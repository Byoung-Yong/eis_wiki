# Note authoring standard

- Use one concept per note.
- Use stable uppercase ASCII IDs.
- Match the canonical title and filename where practical.
- Separate direct observation, processed observation, model estimate, and theoretical construct.
- State equation symbols, units, sign conventions, assumptions, and validity limits.
- Use typed directional relations with valid target IDs.
- Keep source page, chapter, passage, and proprietary location metadata out of public notes.
- Default new notes to `publish: false` until reviewed.
