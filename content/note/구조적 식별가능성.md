# 구조적 식별가능성 (Structural identifiability)

## 핵심 정의

이상적인 무잡음 연속 데이터가 주어졌을 때 모델 매개변수가 수학적으로 유일하게 결정되는 성질이다.

## 개념 이해

모델 구조 자체의 대칭 또는 매개변수 조합 때문에 유일성이 사라질 수 있다.

## EIS에서의 역할

실험 품질과 무관한 모델의 근본적 식별 한계를 평가한다.

## 성립 조건과 실험 해석

관측량과 입력, 모델 매개변수화를 명확히 정의해야 한다.

## 한계와 흔한 오해

실제 데이터에서 잘 적합된다는 사실로 구조적 식별가능성이 증명되지 않는다.
