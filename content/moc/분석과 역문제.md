# 분석과 역문제 (Analysis and inverse problems)

이 영역은 EIS를 이해하는 데 필요한 **분석과 역문제** 개념을 연결한다. 특정 결론이나 단일 해석 순서를 강제하지 않고, 각 노트에서 인접 개념으로 이동할 수 있도록 구성한다.

## 개념 노트 (43)

- [[베이즈 DRT]] — DRT 함수와 잡음에 확률적 사전분포를 두고 사후분포를 추정하는 방법이다.
- [[베이즈 모형 선택]] — 사전분포와 우도를 결합해 후보 모델의 사후 지지도와 매개변수 분포를 비교하는 방법이다.
- [[DRT 커널 함수]] — 완화 시간 $	au$의 단위 기여가 주파수 응답에 어떻게 나타나는지 정의하는 적분 커널 함수가다.
- [[DRT 경계 효과]] — 측정 주파수 범위 바깥의 과정이나 절단된 꼬리가 DRT 시간상수 범위의 가장자리에 인공 피크 또는 왜곡을 만드는 현상이다.
- [[DRT 불확실성]] — 잡음, 유한 대역폭, 정칙화와 초매개변수에 따른 DRT 함수와 피크의 불확실성이다.
- [[DRT 정칙화]] — 비정칙 DRT 역문제에서 잡음 증폭과 해의 요동을 제한하는 추가 벌점 또는 사전분포가다.
- [[DRT 해상도]] — 서로 가까운 완화 시간 기여를 구별하고 피크 위치·폭을 복원할 수 있는 능력이다.
- [[EIS 분포 이동]] — 학습 데이터와 실제 적용 데이터의 재료, 장비, 온도, 주파수 격자 또는 잡음 분포가 달라지는 현상이다.
- [[EIS 기계 학습]] — EIS 스펙트럼 또는 파생 특징에서 회로, 매개변수, 상태, 품질 또는 미래 성능을 학습하는 데이터 기반 방법의 총칭이다.
- [[EIS 데이터 전처리]] — 모델링 전 좌표 변환, 정규화, 필터링, 보간, 정렬 및 이상점 처리를 수행하는 과정이다.
- [[EIS 모델 비교]] — 여러 회로 또는 물리 모델의 적합도, 복잡도, 잔차, 매개변수 안정성과 예측을 비교하는 과정이다.
- [[EIS 모델 식별가능성]] — 임피던스 데이터가 모델 구조와 매개변수를 구별할 수 있는 정도이다.
- [[EIS 잔차 가중]] — 주파수 및 복소 성분별 잔차가 목적함수에 기여하는 상대적 크기를 정하는 규칙이다.
- [[EIS 매개변수 회귀]] — 스펙트럼에서 회로·물리 매개변수 또는 상태량을 직접 예측하는 ML 작업이다.
- [[L-곡선]] — 데이터 잔차 노름과 정칙화 노름의 절충을 로그–로그 평면에 표시하는 방법이다.
- [[L1 정칙화]] — 계수 또는 미분의 절대값 합을 벌점로 사용하는 희소 정규화 방법이다.
- [[티호노프 정칙화]] — 해의 크기 또는 미분의 제곱을 벌점로 추가하는 DRT 정규화 방법이다.
- [[사후예측검사]] — 사후분포에서 생성한 예측과 실제 EIS 데이터의 특징을 비교하는 베이즈 모형 점검이다.
- [[과도한 매개변수화]] — 데이터가 지지하는 정보보다 모델의 자유 매개변수가 많은 상태이다.
- [[과적합]] — 모델이 재현 가능한 시스템 응답보다 측정 잡음과 특정 표본의 세부 변동을 학습한 상태이다.
- [[구조적 식별가능성]] — 이상적인 무잡음 연속 데이터가 주어졌을 때 모델 매개변수가 수학적으로 유일하게 결정되는 성질이다.
- [[능동 주파수 선택]] — 현재 데이터와 목표 과제에 가장 정보가 큰 다음 주파수를 선택하는 적응형 측정 전략이다.
- [[등가회로 분류]] — EIS 데이터에서 사전에 정의된 후보 회로 구조 분류를 예측하는 작업이다.
- [[물리 유도형 EIS 기계 학습]] — 회로식, 인과성, 비음수성, 편미분방정식 또는 알려진 불변성을 학습 구조와 손실에 포함하는 ML 접근이다.
- [[복소 비선형 최소제곱법]] — 복소 임피던스의 실수부와 허수부 잔차를 동시에 최소화하여 모델 매개변수를 추정하는 방법이다.
- [[분포 외 탐지]] — 입력 EIS가 학습 데이터의 분포와 충분히 다른지 판정하는 방법이다.
- [[실용적 식별가능성]] — 유한 대역, 잡음, 표본화와 실제 실험 조건에서 매개변수를 충분히 정밀하게 추정할 수 있는 정도이다.
- [[예측 불확실성 보정]] — 예측 신뢰도 또는 예측 구간의 빈도가 실제 오류율과 일치하도록 평가·보정하는 과정이다.
- [[완화 시간 분포]] — 임피던스를 연속적인 RC 완화 시간 기여의 적분으로 표현하고 그 분포를 역산하는 방법이다.
- [[이산 DRT]] — 연속 DRT 적분을 유한한 기저와 시간상수 격자로 근사한 선형 역문제이다.
- [[일반화 교차검증]] — 선형 평활기의 유효 자유도를 이용해 하나씩 제외한 예측 오차를 근사하는 방법이다.
- [[잔차 분석]] — 측정–모델 차이의 크기, 주파수 구조, 상관과 분포를 점검하는 분석이다.
- [[정칙화 매개변수]] — 데이터 적합과 평활성 또는 희소성 벌점의 상대적 가중치 $\lambda$이다.
- [[정보 기준]] — 모델 적합도와 매개변수 수를 함께 고려해 후보 모델을 비교하는 통계 지표의 총칭이다.
- [[제외 대역 예측]] — 적합에 사용하지 않은 주파수 구간에 대한 모델 예측이다.
- [[주파수 대역 삭제 검증]] — 연속된 주파수 구간을 적합에서 제외하고 모델이 그 구간을 예측하는지 평가하는 방법이다.
- [[초기 매개변수]] — 비선형 최적화를 시작할 때 지정하는 매개변수 값이다.
- [[매개변수 경계]] — 최적화에서 매개변수가 허용되는 범위를 제한하는 조건이다.
- [[매개변수 공분산]] — 추정 매개변수의 분산과 선형 상관을 근사하는 행렬이다.
- [[매개변수 상관]] — 모델 매개변수 변화가 서로 보상되어 비슷한 스펙트럼을 만드는 성질이다.
- [[매개변수 신뢰구간]] — 반복 표본 또는 근사 통계 모델에서 추정 매개변수의 불확실 범위를 나타낸다.
- [[프로파일 우도]] — 관심 매개변수를 고정하고 나머지를 재최적화해 목적함수 또는 우도 변화를 추적하는 방법이다.
- [[회로 구조의 비고유성]] — 서로 다른 회로 구조가 유한 주파수 범위에서 같거나 매우 유사한 임피던스를 생성할 수 있는 성질이다.
