# 측정과 계측 (Measurement and instrumentation)

이 영역은 EIS를 이해하는 데 필요한 **측정과 계측** 개념을 연결한다. 특정 결론이나 단일 해석 순서를 강제하지 않고, 각 노트에서 인접 개념으로 이동할 수 있도록 구성한다.

## 개념 노트 (26)

- [[2전극 EIS]] — 같은 두 전극이 전류 전달과 전압 측정을 담당하는 셀 구성이다.
- [[3전극 EIS]] — 작업 전극, 상대 전극, 기준 전극을 분리한 셀 구성이다.
- [[4전극 EIS]] — 전류 인가 전극과 전압 감지 전극을 분리한 구성이다.
- [[EIS 셀 구성]] — 전극 수, 배치, 전류 경로, 전압 감지 위치를 포함하는 측정 기하 구조이다.
- [[EIS 측정 시간]] — 선택한 모든 주파수에서 응답을 수집하는 데 걸리는 총 시간이다.
- [[개방회로 EIS]] — 외부 직류 전류가 거의 없는 개방회로 전위 부근에서 수행하는 EIS이다.
- [[계측기 대역폭]] — 장비가 지정된 정확도와 안정성으로 제어·증폭·측정할 수 있는 주파수 범위이다.
- [[광대역 자극]] — 여러 주파수 정보를 한 시간 구간에서 얻도록 설계한 자극의 총칭이다.
- [[기준 전극]] — 알려지고 안정한 전위를 제공하여 작업 전극 전위를 측정하는 전극이다.
- [[기준 전극 위치]] — 기준 전극 감지점이 작동·상대 전극과 전류 경로에 대해 놓인 위치이다.
- [[다중 사인 EIS]] — 여러 주파수 성분을 동시에 포함한 파형으로 임피던스를 추정하는 방식이다.
- [[단일 사인 EIS]] — 한 번에 하나의 주파수 사인파를 순차적으로 가하는 측정 방식이다.
- [[상대 전극]] — 작업 전극 전류를 완성하는 반대쪽 전극이다.
- [[창 함수 적용]] — 유한 시간 기록의 경계 불연속을 줄이기 위해 신호에 가중 창을 적용하는 처리이다.
- [[계의 변화 시간척도]] — 측정 중 온도, 전위, 조성, 계면 구조 또는 열화 상태가 의미 있게 변하는 특성 시간이다.
- [[작업 전극]] — 관심 전기화학 반응이 조사되는 전극이다.
- [[전류 제어 EIS]] — 작동점 주변에 교류 전류를 가하고 전압 응답을 측정하는 EIS 방식이다.
- [[전위 제어 EIS]] — 작동점 주변에 교류 전압을 가하고 전류 응답을 측정하는 EIS 방식이다.
- [[정전위기]] — 작업 전극 전위를 기준 전극에 대해 제어하고 전류를 측정하는 장비이다.
- [[정전류기]] — 전류를 제어하고 그에 따른 전압을 측정하는 장비 또는 제어 모드이다.
- [[주기 수와 적분 시간]] — 각 주파수에서 분석에 사용하는 반복 주기 수와 평균 시간이다.
- [[주파수 격자]] — 측정할 주파수 범위와 각 10배 주파수 구간당 점 수를 정한 표본화 계획이다.
- [[주파수 응답 분석기]] — 가해진 자극과 응답에서 주파수별 진폭과 위상을 추정하는 계측 또는 알고리즘 구성요소이다.
- [[차폐와 접지]] — 외부 전자기 간섭과 접지 루프를 줄이기 위한 배선 및 실험 구성이다.
- [[측정 전 평형화]] — EIS 시작 전 작동점에서 시스템이 충분히 안정되도록 기다리는 절차이다.
- [[케이블과 측정 지그]] — 셀과 계측기를 연결하는 케이블, 커넥터, 홀더 및 측정 지그의 전기적 네트워크이다.
